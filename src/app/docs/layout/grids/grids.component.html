Columns
- Columns divide horizontal space into indivisible units called "column". All column in a Columns must specify their width as proportion of the total available row width.
- Each column will have an equal width, no matter the number of columns if the width is not specified.
How it's Responsive?
By default, columns are only activated from tablet onwards. This means columns are stacked on top of each other on mobile.

How Many Columns/Row?
All grid systems chooses an arbitrary column count to allow per row. GlobalUI's default theme uses 12 columns.
Note it?
You can add any number of column to the row, but column width should not increase more than 12 per row, if increases it will not be wrapped to next row by default. You can also wrap the row which you can see below.

Mobile columns
- By default, columns are only activated from tablet onwards. This means columns are stacked on top of each other on mobile.
- If you want columns to work on mobile too, just add the grow-mobile modifier on the columns container

Responsiveness
- Column inside the columns can be grow with a variable size according to the screen widths by a syntax :
grow-screen_type-size

- screen_type can be tablet, desktop, widescreen, fullhd.
- size can variable from 1-12.
In Above example, 1st column width is auto in mobile, 3/12 in tablet, 8/12 in desktop, 4/12 in widescreen, 2/12 in fullhd.
