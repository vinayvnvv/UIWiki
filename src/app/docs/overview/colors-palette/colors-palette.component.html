GlobalUI comes with a two kinds of color scheme.
  • Primary Colors:  These colors are primary to site and can be change or add with sass_vars or global-ui-build tools .
  • Fixed Colors Palette:  These colors are fixed by GlobalUI and can be optional to include for your theme.

  • GlobalUI comes with 7 primary site colors.
  • You can add or edit or remove the colors you want to like by global-ui-build-tools.
    Steps to customize the primary site colors:
  • Open custom.json by navigating global-ui > customize > custom.json.
  • In the file add vars field to the main object if it's not there. Then add colors field to the modules field if it's not there.
    It will look like:
    { "vars": { "colors": [ {"primary": "rgb(4, 131, 169)"}, {"info": "hsl(217, 71%, 53%)"}, {"success": "hsl(141, 71%, 48%)"}, {"warning": "hsl(48, 100%, 67%)"}, {"danger": "hsl(348, 100%, 61%)"}, {"light": "hsl(0, 0%, 96%)"}, {"dark": "hsl(0, 0%, 21%)"} ] } }
  • Now run gulp build command in the terminal. finally it will generate the final css file with specified colors to the dist folder.
  • You can also add additional colors to your primary site colors.
    Example: If you want to add secondary color, it will just look like:
    { "vars": { "colors": [ {"primary": "rgb(4, 131, 169)"}, {"secondary": "hsl(217, 33%, 31%)"} {"info": "hsl(217, 71%, 53%)"}, {"success": "hsl(141, 71%, 48%)"}, {"warning": "hsl(48, 100%, 67%)"}, {"danger": "hsl(348, 100%, 61%)"}, {"light": "hsl(0, 0%, 96%)"}, {"dark": "hsl(0, 0%, 21%)"} ] } }
  • If extra_colors field is not there in modules filed , it will take all the colors from the palette. So that you can add all the colors by removing the extra_colors field.
  • By default global-ui-build-tools supports all pre-defined sass functions so that you can specify the colors using sass function.

    Example:
    { "vars": { "colors": [ {"primary": "lighten(#342445, 20%)"}, ] } }

  • All colors in the Fixed Colors Palette included in GlobalUI by default.
  • You can select only required colors you want to like by global-ui-build-tools so that you can decrease the size of your theme css file.
    Steps to select required colors to your theme:
  • Open custom.json by navigating global-ui > customize > custom.json.
  • In the file add modules field to the main object if it's not there. Then add extra_colors field to the modules field.
    It will look like:
    { "modules": { "extra_colors": ["red", "blue", "blue-grey"] } }
  • Now run gulp build command in the terminal. finally it will generate the final css file with specified colors to the dist folder.
  • You can add any number of color by specifying it's name from below palette table.
  • If you dont want any extra colors , just give empty array to extra_colors field.
    { "modules": { "extra_colors": [] } }
  • If extra_colors field is not there in modules filed , it will take all the colors from the palette. So that you can add all the colors by removing the extra_colors field.
